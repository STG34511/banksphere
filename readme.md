# BankSphere

Enterprise digital banking platform built using Spring Boot and React.

## Features

- Customer onboarding
- Officer approval workflow
- Account management
- Beneficiary management
- Internal transfers
- External transfers
- Ledger-based accounting
- Regulatory screening
- Refund processing
- Audit trails
