# <a name="banksphere---delivery-roadmap"></a>BankSphere - Delivery Roadmap
## <a name="vision"></a>Vision
Deliver a deployable banking application early and evolve it through incremental iterations.

Every iteration must result in a working application.

-----
# <a name="iteration-1---foundation-mvp"></a>Iteration 1 - Foundation (MVP)
Goal:

Customer onboarding and authentication.

Features:

- Registration
- Officer review
- Officer approval
- Account creation
- Email notification
- Login
- Customer dashboard
- Balance display
- Recent transactions

Status:

Deployable

-----
# <a name="iteration-2---core-banking"></a>Iteration 2 - Core Banking
Goal:

Basic money movement.

Features:

- Beneficiary management
- Internal transfers
- Ledger integration
- Transaction history
- Balance updates

Status:

Deployable

-----
# <a name="iteration-3---external-transfers"></a>Iteration 3 - External Transfers
Goal:

Multi-rail payments.

Features:

- IMPS
- NEFT
- RTGS
- Transfer processing workflow
- Transaction states

Status:

Deployable

-----
# <a name="iteration-4---regulatory-engine"></a>Iteration 4 - Regulatory Engine
Goal:

Compliance processing.

Features:

- Daily limits
- Account status checks
- OFAC screening
- Officer review queue
- Transaction approval workflow

Status:

Deployable

-----
# <a name="iteration-5---refunds-enquiry"></a>Iteration 5 - Refunds & Enquiry
Goal:

Operational resilience.

Features:

- Transaction enquiry
- Refund processing
- Scheduled refund jobs
- Refund notifications

Status:

Deployable

-----
# <a name="iteration-6---operations"></a>Iteration 6 - Operations
Goal:

Administrative controls.

Features:

- Audit trail
- Account freeze
- Account closure
- Operational dashboards
- Reporting

Status:

Production-ready portfolio project
